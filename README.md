# 孕宝助手 PWA

## 网址
- **Cloudflare Pages：** https://bbmmabb.pages.dev （国内推荐）
- **GitHub Pages：** https://mmayi3344.github.io/bbmmabb/

## 技术栈
- 纯前端 PWA（HTML + CSS + JS）
- 数据存储在浏览器 localStorage（每人独立，互不可见）
- Service Worker 离线缓存
- GitHub Pages + Cloudflare Pages 双托管

---

## 功能模块

### 🏠 首页
- 预产期倒计时（天）+ 可修改预产期
- 40 周进度网格（已完成/当前深绿高亮）
- 距离上次产检间隔（自动取最新一条产检记录）
- 每周孕期小贴士

### 🤰 孕期
- **🦶 胎动计数** — 支持 3/5/10 分钟合并窗口，有效胎动判断，历史记录
- **⚖️ 孕期体重** — 每日记录，单条删除 / 清空全部
- **🏥 产检提醒** — 项目 + 日期提醒清单
- **📋 产检记录** — 日期 + 项目 + 拍照/选图，支持 ✏️ 修改和单条删除

### 👶 宝宝
- **🍼 哺乳计时** — 左侧/右侧分别计时，历史记录
- **🧷 尿布记录** — 便便/嘘嘘/都有
- **📏 生长记录** — 体重 + 身长
- **😴 睡眠记录** — 睡眠时长

### 💉 疫苗
- 国家标准接种时间表（出生~6 岁），点击打勾标记已完成

### 📦 其他
- **🎒 待产包** — 项目名称 + 数量 + 单价，自动计算小计和总价
- **📝 笔记** — 孕期日记 / 育儿笔记

---

## 通用功能
- ✏️ 标题可点击自定义
- 🗑 所有记录支持单条删除 + 一键清空全部（带确认）
- 📤📥 数据导出/导入（换手机迁移：旧手机导出 → 新手机导入）
- 📲 可添加到手机主屏幕，像原生 App 一样使用
- 🌙 深色模式兼容

---

## 更新方法
1. 修改代码后提交并推送
2. GitHub Pages 自动部署（~1 分钟）
3. Cloudflare Pages 自动同步
4. 手机强制刷新清除缓存

---

## 强制刷新
- iPhone Safari：地址栏 → aA → 重新载入网站
- Android Chrome：地址栏 → 🔒 → 网站设置 → 清除并重置
